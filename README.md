## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/TASM-Solutions.git
```

Go to project folder

```bash
cd TASM-Solutions
```

Install dependencies

```bash
composer install
```

Copy environment file

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Configure database in `.env`

Run migrations

```bash
php artisan migrate
```

Start development server

```bash
php artisan serve
```
